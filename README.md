This program prompts the user to enter a numerical grade between 0 and 100 and displays the corresponding letter grade.

Usage

1. Run the program
2. Enter a numerical grade between 0 and 100 when prompted
3. The program will display the corresponding letter grade

Letter Grade Scale

- 90 and above: A
- 80-89: B
- 70-79: C
- 60-69: D
- Below 60: F

Example Output


Enter a numerical grade: 85
Letter Grade: B


Technical Details

- Programming Language: Java
- Input Validation: Checks for numerical input between 0 and 100
